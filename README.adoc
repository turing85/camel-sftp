= Camel SFTP: password logged
:author: Marco Bungart

== Reproducer
.Building the application
[source, bash]
----
./mvnw clean package
----

.Starting JMS broker
[source, bash]
----
docker-compose \
  --file local-deployment/docker-compose.yml \
  up --detach
----

.Starting the application
[source, bash]
----
java -jar target/camel-sftp-1.0-SNAPSHOT-executable-jar.jar
----
